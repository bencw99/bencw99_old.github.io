Below you can download four games I programmed in the Java programming language called MoonLander, MarsLander and Adventure Game.

To play them, click on the links below, extract their files from the zip files downloaded, and click on the file called Game.jar. Make sure to keep the file named Game.jar in the same folder as the images and files, or else the games won't work.

I am still working on the Adventure Game. For now, a user can choose an avatar, walk around with the arrow keys, pick up items with d, and use items with the space bar.

The moonlander game is completed and was the first game I made in java. The user can control a rocket using the arrow keys and change various attributes of the rocket and controls in the settings page

The mars lander game is a version of the moonlander game with trickier terrain

Finally, my newest project is the checkers game. To move, click on the piece you want to move, and all of the locations through which you want to move (both the final location and the loaction between two ennemies for a double jump, for example) then press 'r' to register the move. If you wish to restart the piece selection process press u'.

I will be shortly updating both Moon Lander, and Mars Lander and will merge them both into one game with better graphics, and a better graphicsl user interface
